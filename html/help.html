FAQ

What is txDesigner?
txDESIGNER is an intuitive application for visualizing the study design during the protocol design, study build and during submission phase, streamlining the clinical trial process and helping life sciences organizations optimize their R&D investments and time.
How txDESIGNER works?
txDESIGNER maps the Visit, Form, Field and the corresponding validation details from ALS and generates the CRFs in a PDF format which can be used as specification for trial build, regulatory submission and archival purpose.
To what extent it is going to help users?
Makes the protocol development visual and therefore more interactive
  • The clinical science team gets to visualize the actual study prior to being built, also eliminates the need to learn EDC platform
  • Protocol development and design phases become visual, offers great deal of clarity during the trial realization phase thus reducing the time span
  • Address the collaboration challenges during the protocol writing phase in a geographically distributed teams
What is ALS File?
ALS (Architect Loader Spreadsheet - "Rave output file") is a Microsoft excel file, which contains the definition of a study. Using the Architect Loader Spreadsheet a trial can be edited and archived in an offline spreadsheet format.
Why we need to pick Matrix Sheet?
This feature facilitates users to visualize the eCRF in a consolidated matrix view by Domain and Visit and offer an overview of the overall trial design.

What is blank PDF?
Blank PDF is a blue print of the eCRF that is generated by extracting the specifications provided in the ALS (Architect Loader Spreadsheet). This will help the user visualize how the EDC screens would appear once they would be designed in the EDC system.
What is Edit Check PDF?
This is an annotated PDF along with the corresponding data validation details. This PDF output can be handed over to the EDC programmer to program the edit checks of the study.
What is Annotated PDF?
This is a blank annotated PDF output that documents the location of the data with the corresponding names of the datasets and the names of those variables included in the submitted datasets.
What does "Run Log" do?
Run Log stores all the previously generated PDF along with uploaded corresponding ALS (input) files, so they can be accessed at any time. Please note that Run Log is user specific and user can access only those files that were uploaded and generated by him/her.
Trouble uploading the ALS file?
Prior to uploading the ALS file please make sure that the file is in Excel 97-2003 format and the file name should be less than 35 characters.

Does the application require any additional software to run?
No additional software except Internet Explorer v8.0 (and above) or Google Chrome is required to operate txDesigner.

How can I access the files that I generated in the past?
Run Log maintains and stores the previous PDF output that were generated by the user and can be accessed anytime.

How can I get access to the txDesigner?
Please reach out to your Org Admin user to create your account and provide access to txDesigner.

How frequently do I need to change my password?
Password change notification is flashed to the user upon login, a week prior to the completion of 90 days from the last time the password was changed. However, the user can wish to change the password at anytime using the “Change Password” option once they are logged into txDesigner.
What security measures are in place to ensure that no unauthorized users have access to the application?
txDesigner offers multi-level password protection, secure log-in and a secure server. txDesigner also comes with standard user access privilege levels and the ability to create customized user access privileges. This allows our clients to assign the appropriate access level to each person involved in a study.
